## 考试回忆
试卷构成
* 20 分 单选题
* 10 分 不定项选择题
* 70 分 主观题（其中，辨析题要先给结论再给辨析，答案错误直接没分）

21-22秋冬大题考的基本都是习思想，也就是毛概的后半本书。
